# My Portfolio

A personal portfolio website built with React, TypeScript, and Vite to showcase projects, skills, and profile information.

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS

## Features

- Responsive portfolio layout
- Project and profile content sections
- Modern UI components and icons
- Fast local development workflow with Vite

## Getting Started

### Prerequisites

- Node.js 18+
- npm

### Install Dependencies

```bash
npm install
```

### Run in Development

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Project Structure

- `components/` - reusable UI components
- `data/` - portfolio data/content files
- `styles/` - style definitions
- `App.tsx` - main app entry component
- `main.tsx` - React bootstrap
